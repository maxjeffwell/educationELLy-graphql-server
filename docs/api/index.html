EducationELLy GraphQL API

GraphQL API for the EducationELLy English Language Learner Management System.

Overview

This API provides endpoints for managing students, user authentication, and AI-powered educational features.

Authentication

Most operations require authentication via JWT token. Include the token in the x-token header:

x-token: <your-jwt-token>

Getting Started

  1. Sign up for an account using the signUp mutation
  2. Sign in to receive your JWT token using the signIn mutation
  3. Use the token in subsequent requests

Error Handling

Errors are returned in standard GraphQL format with an errors array containing error messages and codes.

Description

Authenticate an existing user. Sets an httpOnly authentication cookie and returns success status. The cookie is automatically sent with subsequent requests when using credentials: 'include'.

Response

Returns an AuthPayload!

Arguments
Name Description
login - String! Email address used during registration
password - String! Account password

Example

Query
mutation signIn(
  $login: String!,
  $password: String!
) {
  signIn(
    login: $login,
    password: $password
  ) {
    success
    user {
      ...UserFragment
    }
  }
}

The ID scalar type represents a unique identifier, often used to refetch an object or as key for a cache.
